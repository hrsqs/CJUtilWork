# CJUtilWork (1.0.1)

- 说明：包含常用的宏定义、常用类的分类

## 安装说明

方法一（下载）</br>
- 下载GitHub源文件，拷贝Data文件夹下的所有文件至您的项目工程中。

方法二（CocoaPods）</br>
```ruby
platform :ios, '8.0'

target 'TargetName' do
pod 'CJUtilWork'
end
```

## 使用方法

```objective-c
import "CJUtil.h"
```
建议写在pch文件里
